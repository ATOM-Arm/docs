Introdução
Nosso grupo de estudantes do curso de Ciência da Computação concebeu o projeto de desenvolver um braço robótico utilizando conhecimentos adquiridos nas disciplinas de Robótica, Algoritmos e Estruturas de Dados. O objetivo deste projeto é aplicar conceitos teóricos em um projeto prático, integrando a automação com visão computacional. A ideia central é criar um braço robótico que possa replicar os movimentos de uma mão humana, controlado por meio de algoritmos de visão computacional implementados em Python utilizando a biblioteca cvzone.

Objetivos
Objetivo Geral: Desenvolver um braço robótico que replique os movimentos de uma mão humana utilizando visão computacional.
Objetivos Específicos:
Integrar a impressão 3D para criação das peças do robô.
Implementar um sistema de visão computacional para rastreamento dos movimentos da mão.
Programar o controle do braço robótico utilizando a biblioteca cvzone em Python.
Testar e ajustar o sistema para garantir precisão e funcionalidade.
